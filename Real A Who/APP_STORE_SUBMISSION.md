# App Store Submission Checklist

This project is set up as a simple offline reflection journal that is real, functional, and reviewable without a login.

## What is already in the app

- Daily prompt with a clear core action
- Local note creation, editing, deletion, favorites, and search
- On-device persistence with no account and no backend dependency
- Insights screen built from saved entries
- About screen that explains privacy and functionality
- `ITSAppUsesNonExemptEncryption = NO` for simpler export-compliance handling

## Still required before you submit

- Replace the temporary app name, subtitle, and description in App Store Connect if you want different branding
- Add final screenshots from the iPhone and iPad simulator
- Add a support URL in App Store Connect
- Fill out App Privacy in App Store Connect as `Data Not Collected` if that remains true
- Write accurate Review Notes explaining that the app works fully offline and does not require an account
- Test on a physical device before upload

## Suggested App Review Notes

`Real A Who` is a fully offline reflection journal. App Review can access the complete experience immediately on launch with no login, subscription, or external service. All entries are stored locally on-device.

## Metadata direction

- Category: `Lifestyle`
- Age rating: `4+`
- Core description: private daily reflection journal with prompts, saved entries, favorites, and local insights

## Current review-guideline strategy

- Avoid placeholder or unfinished flows
- Avoid login, payments, broken links, or hidden functionality
- Keep all metadata honest and matched to the actual app
