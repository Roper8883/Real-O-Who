# User Flows

## Seller listing flow

1. Create account and verify email.
2. Complete profile and identity/authority declarations.
3. Start listing draft and choose property type.
4. Enter address and state/territory.
5. Complete state-aware disclosure checklist.
6. Upload media and documents.
7. Set pricing, inspection availability, and seller notes.
8. Preview listing with disclosure status and publish blockers.
9. Publish or route to compliance review.

## Buyer property journey

1. Browse map/list search.
2. Save properties and create folders.
3. Review listing detail, documents, and comparable sales.
4. Message owner or view pinned FAQs.
5. Book open-home or private inspection.
6. Request seller-provided or third-party building/pest report.
7. Submit structured offer with conditions and clear non-binding warning.
8. Move to contract request and external legal review if accepted in principle.

## Admin/compliance flow

1. Review newly created or flagged listings.
2. Check seller verification, media, and disclosure bundles.
3. Moderate reported messages or suspicious offers.
4. Approve providers and credentials.
5. Update jurisdiction rule content and feature flags.
6. Export audit trail where needed for support or dispute handling.
