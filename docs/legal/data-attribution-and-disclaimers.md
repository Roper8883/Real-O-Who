# Data Attribution and Disclaimers

## Provenance labels

Where possible, the platform should label information as:

- seller supplied
- public record
- licensed data provider
- estimated / automated
- unavailable

## No fabrication

The platform must not fabricate sold history, title details, zoning, planning overlays, contamination records, body corporate fees, council rates, or comparable sales.

## Availability disclaimer

Not all data will be available for every property. Missing data should be shown as unavailable rather than guessed.
