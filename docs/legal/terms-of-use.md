# Terms of Use

## Status

Draft for legal review. This document is written for the Homeowner platform, an Australia-first private property sale marketplace and workflow product.

## Service description

Homeowner provides:

- property listing and search tools
- owner-to-buyer communication and workflow coordination
- inspection booking
- building and pest service marketplace features
- document sharing and disclosure assistance
- non-binding offer capture and negotiation workflow

Homeowner does not, in v1, act as:

- a law firm
- a conveyancer
- a licensed real estate agent
- a buyer's agent
- an escrow service
- a trust account holder
- a settlement platform
