# Security Overview

## Principles

- least privilege
- explicit server-side authorization
- privacy by design
- auditable high-risk actions
- safe defaults for legal ambiguity

## Controls in scope

- email verification
- optional MFA scaffolding
- password hashing
- session/device history
- role-aware permissions
- signed file access
- moderation hooks
- rate-limit and bot-protection hooks
- audit logs for sensitive operations
